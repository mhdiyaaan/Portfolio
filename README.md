## 📦 Run Locally

```bash
# Clone the repo
git clone https://github.com/mhdiyaaan/Portfolio.git

# Navigate into the folder
cd Portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.
